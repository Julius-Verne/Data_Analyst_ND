Using this 'bee swarm graph' we can see the distribution of the different samples and the values where each category tends to fall into. The congruent sample tends to have lower values overall with few variance towards the mean. Meanwhile, the incongruent sample has a wider spectrum of values including two outliers: 34.2 and 35.2. Although there are some segments when they categories overlap, both of them tend to have different values.
